# hiivmind-corpus-github

A Claude Code plugin marketplace containing documentation corpora for GitHub products.

## Included Corpora

| Plugin | Description |
|--------|-------------|
| `hiivmind-corpus-github-docs` | Official GitHub documentation (Actions, Copilot, REST API, GraphQL, etc.) |

## Installation

```bash
/plugin marketplace add https://github.com/hiivmind/hiivmind-corpus-github
```

## Usage

After installation, the corpus skills will be available. Ask Claude about GitHub topics:

- "How do I set up GitHub Actions for my project?"
- "What's the GraphQL query to list repository issues?"
- "How do I configure Copilot for my organization?"

## Maintenance

This marketplace is managed by the `hiivmind-corpus` meta-plugin. See CLAUDE.md for details.
