# GitHub Documentation Corpus

A curated documentation corpus for GitHub is available. Use the **hiivmind-corpus-github-docs-navigate** skill when you need to look up GitHub documentation.

## When to Use

- Working with GitHub Actions, workflows, or CI/CD
- Using GitHub REST API or GraphQL API
- Configuring repositories, branch protection, or permissions
- Setting up GitHub Copilot or code security features
- Managing pull requests, issues, or projects
- Need accurate, up-to-date GitHub documentation

## How to Invoke

Say something like:
- "Look up GitHub docs for..."
- "Check the GitHub corpus for..."
- "What does the GitHub documentation say about..."

## Key Topics

- GitHub Actions (workflows, runners, reusable workflows, secrets)
- REST API (authentication, endpoints, pagination)
- GraphQL API (schema, queries, mutations)
- Copilot (setup, usage, extensions)
- Code Security (Dependabot, secret scanning, CodeQL)
- Repositories (settings, branch protection, rulesets)
- Pull Requests and Issues

## Example Questions

- "How do I set up a reusable workflow in GitHub Actions?"
- "What's the GraphQL query to get repository information?"
- "How do I configure branch protection rules?"
- "How do I use GitHub Copilot in VS Code?"
